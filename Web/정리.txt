<태그 tag >

<title>~</title>	~ 가 웹페이지의 제목이 됨. (본문의 제목)
<meta charset = "utf-8">	한글이 깨지지 않도록
<!doctype html>		html 문서임을 선언, 보통 문서의 최상단에 작성 ( 관용적, 헤더파일처럼 생각해도됨)

<html>~</html>		~ 가 html의 코드 영역임을 표시 , 실직ㄹ적으로는 head 와 body tag를 감싸고 있다고 봐도 됨  
 		
<head>~</head>		~ 가 본문과 달리 부가적인 내용(meta, title, style 태그)을 표시

<body>~</body>		~ 가 본문의 내용(header, section, footer, nav, aricle 태그)
 
<strong>~</strong>	~ 을 진하게 표시

<u>~</u>		~ 을 밑줄 표시

<h1>~</h1>		~ 제목을 나타내는 heading 
		h1 ~ h6 까지 있음

HTML New Line Tag
<br>	vs	<p></p>

1. br tag
 강제줄바꿈
 
2. p tag
 하나의 문단 Paragraph 를 의미함

p tag를 사용하는 것을 지향해야 한다.
왜냐하면 p tag가 CSS 와 연동이 되어서 <p style = "width = 400px"> 등의 제한을 할 수 있기 때문.

Img Tag

<img src = "키움 역대 성적.png", width = 60%>
위 예시 코드에는 2가지 속성
src ( img 의 위치 )
width ( 가로 크기 )
속성에 대한 부분은 순서가 바뀌어도 무관
